Кратко про Margin

Чтобы сделать отступ только с одной стороны, используй margin-top (сверху), margin-right (справа), margin-bottom (снизу) или margin-left (слева).

Примеры

Применяется ко всем четырём сторонам:

.p {
margin: 10px;
}

Сверху и снизу | слева и справа:

.p {
margin: 25px 15px;
}

Сверху | слева и справа | снизу:

.p {
margin: 10px 5px 35px;
}

Сверху | справа | снизу | слева:

.p {
margin: 25px 5px 10px 15px;
}

Кратко про Padding

Свойство padding — или внутренний отступ — позволяет оттолкнуть контент от границ родительского элемента. По записи схож с margin

Кроме отступов, с помощью padding можно сделать отзывчивое изображение.
